
## Development (Local) and setup guide

1. Clone this repo
2. Be sure to have `node` installed
3. `yarn`
4. `yarn dev`
5. Open http://localhost:4000/api/v1/

## Production setup guide
1. Clone this repo
2. Be sure to have `node` installed
3. `yarn`
4. `yarn  build`
5. `yarn start`
5. Open http://localhost:5000/api/v1/
